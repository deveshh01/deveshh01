### 🔹 Amazon ML Hackathon — Smart Product Pricing *(Top 3% Global 🏆)*
> *Multimodal ML · NLP · Computer Vision · Ensemble Learning*

Developed a **multimodal ML pipeline** to predict product prices from both textual descriptions and product images. Combined **NLP feature extraction** (TF-IDF, word embeddings) with **CNN image embeddings**, trained ensemble models on 75,000+ samples, and optimized performance under strict compute constraints.

`🔧 Tech:` Python · CNN · NLP · Scikit-learn · PyTorch · Feature Engineering
